# Composite Pattern

## Definition

Composite allows individual objects and groups of objects to be treated uniformly.

## Purpose

Represent hierarchical structures as tree structures.

## Why Use It?

Objects and collections of objects can be handled in the same way.

## Advantages

* Simplifies hierarchy management
* Easy scalability
* Uniform treatment

## Interview Definition

Composite is a structural design pattern that composes objects into tree structures and allows clients to treat individual objects and groups uniformly.
